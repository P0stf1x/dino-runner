# Dino Runner (Chrome) Clone

Done in pure HTML + CSS + Javascript

This project is not intended as a serious game or website, so it won't have any fancy web tricks like packaging all the assets or obfuscating the variables for the sake of lowering code size. 

It in theory fits (at least for now) in 1 RTT (13636 bytes out of 14600 excluding sprites, and 14278 including sprites), but for it to take 1 RTT it'd have to be combined into a single file, and I hate build systems for interpreted languages, so no, it won't be minified, at least not by adding a build step

Also I'm not a web dev, so I've used chatgpt for some help (mainly most of CSS, and HTML/JS interactions)

You can find hosted version at https://prfx.xyz/game/dino_runner/
